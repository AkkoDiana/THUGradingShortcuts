# THUGradingShortcuts
针对[清华大学网络学堂](https://learn.tsinghua.edu.cn)的批作业页面设计的脚本。

批作业的页面很多重复操作都只能通过鼠标点击，例如保存批阅结果，保存成功后的弹窗确认。每批一份作业都得来回移动鼠标点击，非常费事，尤其是连续上百次重复操作。

为了批作业方便就写了这个脚本，给批阅页和批阅前页绑定了一些快捷键，这样就可以用键盘完成主要的操作。亲测来看可以节省不少操作时间。
